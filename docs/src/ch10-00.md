# 第十章 多核支持 - SMP

多个 CPU (核)共享同一个任务调度队列，实现任务并发执行。SpinLock 自旋锁对任务队列提供互斥保护。

### PerCPU 作用

专属于 CPU 的本地存储区域，获得更高的本地缓存效率。

类比：

- TLS: thread-local-storage
- PerCPU: cpu-local-storage
