## 5.2 协作式调度机制与示例

### 5.2.1 协作式调度算法 FIFO 机制

- 任务按照先入先出的顺序被 CPU 执行；

- 当前任务一旦获得执行权，就会一直执行；

- 只有两种情况下，其它任务才能获得执行机会：

  - 当前任务执行完成后退出；

  - 当前任务主动调用 yield_now 让出执行权。
